Typescript

¿Que es?

TypeScript es un lenguaje de programación de tipado estático que se compila a JavaScript. Esto significa que le agregas tipos a tus variables, funciones y objetos, lo que permite al compilador detectar errores en tiempo de desarrollo antes de que se conviertan en problemas en tiempo de ejecución.

Sintaxis

TypeScript extiende la sintaxis de JavaScript, por lo que si ya conoces JavaScript, algunos conceptos fundamentales son:

Variables y tipos

  • Declacaciones

    • let: Declaran variables que pueden reasignarse.
    • const: Constantes que no pueden reasignarse.
    • :tipo: Especifica el tipo de variable.

    Tipos de variables

    • string: Cadena de texto.
    • number: Números (Enteros, flotantes)
    • boolean: Valores booleanos (True, False)
    • Null y Undefinied: Valores especiales
    • any: Tipo de comodín (evitar el uso excesivo).
