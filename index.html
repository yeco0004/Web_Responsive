Consulta #1 Sitio Web Responsive

Typescript

TypeScript es un lenguaje de programación que agrega tipado estático a JavaScript, permitiendo detectar errores en tiempo de desarrollo y evitar problemas en tiempo de ejecución.

Angular

Angular es un framework de código abierto de Google para desarrollar aplicaciones web SPA y PWA, tanto para móviles como para escritorio, muy fácilmente siempre.

Ionic

Ionic es un framework de código abierto basado en Angular que permite desarrollar aplicaciones móviles híbridas utilizando tecnologías como HTML, CSS y JavaScript en la web.
